---
title: 'Tokens'
openapi: 'GET /tokens/{address}'
description: 'Get token information on Solana'
---

<Note>
  Direct `/tokens` endpoint is not available. You must specify a token address in a format `/tokens/{address}` where `{address}` is the token address on Solana blockchain like `So11111111111111111111111111111111111111112`.
</Note>

The API provides two endpoints for token-related data:

<Card title="GET /tokens/{address}" icon="circle-info" href="/api-reference/endpoint/tokens">
  Get detailed latest data about a specific Solana token
</Card>

<Card title="GET /tokens/{address}/pools" icon="water" href="/api-reference/endpoint/pools">
  View pools documentation to get liquidity pools data for a specific token
</Card>


### Path Parameters

<ParamField path="address" type="string" required>
  The unique identifier `address` of the token on Solana blockchain
</ParamField>

### Response Fields

<ResponseField name="id" type="string">
  Token's contract address
</ResponseField>

<ResponseField name="name" type="string">
  Token name
</ResponseField>

<ResponseField name="symbol" type="string">
  Token symbol
</ResponseField>

<ResponseField name="chain" type="string">
  Blockchain network (always "solana")
</ResponseField>

<ResponseField name="decimals" type="integer">
  Token decimals
</ResponseField>

<ResponseField name="summary" type="object">
  <Expandable title="Token statistics">
    <ResponseField name="main_pool" type="string">
      Main liquidity pool address
    </ResponseField>
    <ResponseField name="price" type="number">
      Current token price
    </ResponseField>
    <ResponseField name="price_usd" type="number">
      Current token price in USD
    </ResponseField>
    <ResponseField name="liquidity_usd" type="number">
      Total liquidity in USD
    </ResponseField>
    <ResponseField name="24h" type="object">
      24-hour trading statistics
    </ResponseField>
    <ResponseField name="6h" type="object">
      6-hour trading statistics
    </ResponseField>
    <ResponseField name="1h" type="object">
      1-hour trading statistics
    </ResponseField>
  </Expandable>
</ResponseField>


### Usage Example

```bash
curl -X GET "https://api-beta.dexpaprika.com/tokens/So11111111111111111111111111111111111111112"
```

<Note>
  The token address must be a valid Solana token address. For example, Wrapped SOL token address is So11111111111111111111111111111111111111112
</Note>
