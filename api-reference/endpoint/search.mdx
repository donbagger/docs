# Search API

Perform a search query to find tokens, liquidity pools, and decentralized exchanges.

## Parameters

<ParamField query="query" type="string" required>
  Search term to match against token names, pool addresses, or DEX identifiers
</ParamField>

## Response

<ResponseField name="tokens" type="Token[]">
  Array of matching tokens with their metadata and market data
</ResponseField>

<ResponseField name="pools" type="Pool[]">
  Array of matching liquidity pools with their trading information
</ResponseField>

<ResponseField name="dexes" type="Dex[]">
  Array of matching DEXes with their volume and transaction data
</ResponseField>

## Usage Example

```bash
curl -X GET "https://api-beta.dexpaprika.com/search?query=usdc"
```

<Note>
  The search is case-insensitive and will return empty arrays for categories with no matches.
</Note>
