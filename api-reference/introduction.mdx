## Quick Start

```bash
curl -X GET "https://api-beta.dexpaprika.com/networks/solana/tokens/So11111111111111111111111111111111111111112"
```

This will return current data about Wrapped SOL (SOL):

```json
{
  "id": "So11111111111111111111111111111111111111112",
  "name": "Wrapped SOL",
  "symbol": "SOL",
  "chain": "solana",
  "price_usd": 195.22,
  "summary": {
    "volume_usd": 9672251889.53,
    "liquidity_usd": 26001938057.05
  }
}
```

## Base URL

All API endpoints use the following base URL:

```
https://api-beta.dexpaprika.com/networks/solana
```


## Common Use Cases

Here are some popular ways to use our API:

<AccordionGroup>
  <Accordion title="Track Token Prices">
    ```bash
    # Get USDC price and trading data
    curl -X GET "https://api-beta.dexpaprika.com/networks/solana/tokens/EPjFWdd5AufqSSqeM2qN1xzybapC8G4wEGGkZwyTDt1v"
    ```
  </Accordion>
  <Accordion title="Monitor Liquidity Pools">
    ```bash
    # Get all pools where SOL is traded
    curl -X GET "https://api-beta.dexpaprika.com/networks/solana/tokens/So11111111111111111111111111111111111111112/pools"
    ```
  </Accordion>
  <Accordion title="Search Tokens">
    ```bash
    # Search for "Jupiter" token
    curl -X GET "https://api-beta.dexpaprika.com/networks/solana/search?query=jupiter"
    ```
  </Accordion>
</AccordionGroup>

## Rate Limits

The API is currently in beta and free to use with a rate limit of:
- 1 request per second
- 10,000 requests per day
